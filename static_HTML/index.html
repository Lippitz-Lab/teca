Skript zur Vorlesung 'Messmethoden'

Dies ist das interaktive Skript zur Vorlesung 'Messmethoden' im Sommersemester 2022 an der Universität Bayreuth. Es gibt den Inhalt interaktiv oder als Quelltext. Inspririert ist das alles von Computational Thinking am MIT. Die Interaktivität stammt von Pluto.jl basierend auf Julia.

Wenn Sie dem Kurs an der Universität Bayreuth folgen, dann finden Sie organisatorische Hinweise im zugehörigen elearning-Kurs.
